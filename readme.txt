Pour récuperer ce tutoriel, utilisez les commandes suivantes : 
git init
git clone https://github.com/jeanbaptistedalle/jdbc.git

Une fois ceci fait, il vous faudrat initialiser votre base de données. Pour cela, il vous faudra créer une base nommée 'jdbc' et posséder un utilisateur nommé 'root' et dont le mot de passe est 'root' (ou modifier la méthode startConnection se trouvant dans le main)
